# check_apc_ups
APC UPS Nagios plugin

## Usage

### Sintax
``` bash
check_apc_ups <IP_ADDR> <COMMUNITY> <RESOURCE> <WARNING> <CRITICAL>
```

### Example
``` bash
check_apc_ups 192.168.0.2 public oVOLT 225 200
```
